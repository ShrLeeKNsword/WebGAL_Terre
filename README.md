# WebGAL_Terre

**重新定义Galgame的制作方式**

以最快捷的方式创建属于你自己的 Galgame，并支持导出为网页和 Windows 可执行文件。

方便地上传、管理和预览你的游戏素材。

多标签页的编辑器，可以让你快速在多个剧本间切换。
